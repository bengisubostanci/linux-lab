EN 🇬🇧
What is Linux?: An open-source, free operating system (OS) that manages hardware resources.

Kernel vs. Distribution: Linux is strictly the kernel (core engine). Ubuntu, CentOS, Debian, and RedHat are distributions (distros) built on top of it.

GNU & Free Software: Created by Richard Stallman to guarantee user freedoms (use, study, modify, distribute). GNU is Not UNIX.

The Shell: A program that takes keyboard commands and passes them to the OS. bash = Bourne Again Shell.

Command Syntax: command -options arguments (e.g., ls -l /etc)

TR 🇹🇷
Linux Nedir?: Donanım kaynaklarını yöneten, açık kaynaklı ve ücretsiz bir işletim sistemidir.

Çekirdek (Kernel) vs. Dağıtım (Distro): Linux aslında sadece çekirdektir (merkezi motor). Ubuntu, CentOS, Debian ve RedHat ise bu çekirdeği kullanan dağıtımlardır.

GNU & Özgür Yazılım: Richard Stallman tarafından kullanıcılara yazılımı kullanma, inceleme, değiştirme ve dağıtma özgürlüğü sunmak için başlatılmıştır. GNU is Not UNIX.

Shell (Kabuk): Klavyeden girilen komutları alıp işletim sistemine ileten programdır. bash = Bourne Again Shell.

Komut Yapısı: komut -opsiyonlar argümanlar (Örn: ls -l /etc)
